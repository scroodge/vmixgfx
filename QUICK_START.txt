vMix Score Control - Quick Start Guide
Version: 0.1a (Alpha)
=======================================

INSTALLATION:
-------------
1. Extract vmix_score_control.exe to any folder
2. Double-click vmix_score_control.exe to start
3. A console window will appear showing server status

USAGE:
------
Once the server is running:

Control Panel (for managing scores):
  http://localhost:8000/control

Overlay (for vMix Browser Input):
  http://localhost:8000/overlay?matchId=1

NETWORK ACCESS:
---------------
The server can be accessed from other devices on your network:
  http://YOUR_IP_ADDRESS:8000/control

Example:
  If your computer's IP is 192.168.1.100, access:
  http://192.168.1.100:8000/control

vMIX CONFIGURATION:
-------------------
1. Open vMix and click "Add Input"
2. Select "Browser" as input type
3. Set URL to: http://localhost:8000/overlay?matchId=1
   (Or use your computer's IP for network access)
4. Set Width: 1920, Height: 1080
5. Enable "Transparent Background"
6. Set Update Rate: 60 FPS (recommended)

STOPPING THE SERVER:
--------------------
- Click the X button on the console window, or
- Press Ctrl+C in the console window

TROUBLESHOOTING:
----------------
Port 8000 already in use:
  - Close any other instances of the application
  - Or restart your computer

Can't access from network:
  - Windows Firewall may prompt for permission (click Allow)
  - Make sure other devices are on the same network
  - Check your computer's IP address in network settings

vMix shows blank overlay:
  - Make sure the server is running (check console window)
  - Try accessing the URL directly in a browser first
  - Check that "Transparent Background" is enabled in vMix

For more information, see BUILD.md or README.md

vMix Score Control - Хуткі даведнік па старце
Версія: 0.1a (Альфа)
===============================================

УСТАНОЎКА:
----------
1. Распакуйце vmix_score_control.exe ў любую папку
2. Двойчы націсніце vmix_score_control.exe для запуску
3. З'явіцца акно кансолі, якое паказвае статус сервера

ВЫКАРЫСТАННЕ:
-------------
Пасля таго, як сервер запусціўся:

Панэль кіравання (для кіравання лікам):
  http://localhost:8000/control

Оверлей (для vMix Browser Input):
  http://localhost:8000/overlay?matchId=1

СЕТКАВЫ ДОСТУП:
---------------
Сервер можа быць даступны з іншых прылад у вашай сетцы:
  http://ВАШ_IP_АДРАС:8000/control

Прыклад:
  Калі IP-адрас вашага камп'ютара 192.168.1.100, доступ:
  http://192.168.1.100:8000/control

НАЛАДКА vMIX:
-------------
1. Адкрыйце vMix і націсніце "Add Input"
2. Выберыце "Browser" як тып уводу
3. Усталюйце URL на: http://localhost:8000/overlay?matchId=1
   (Або выкарыстоўвайце IP-адрас вашага камп'ютара для сеткавага доступу)
4. Усталюйце Шырыню: 1920, Вышыню: 1080
5. Уключыце "Transparent Background"
6. Усталюйце Хуткасць абнаўлення: 60 FPS (рэкамендуецца)

СПЫНЕННЕ СЕРВЕРА:
-----------------
- Націсніце кнопку X на акне кансолі, або
- Націсніце Ctrl+C у акне кансолі

ВЫРАШЭННЕ ПРАБЛЕМ:
------------------
Порт 8000 ужо выкарыстоўваецца:
  - Зачыніце любыя іншыя экзэмпляры дадатка
  - Або перазапусціце камп'ютар

Немагчыма атрымаць доступ з сеткі:
  - Windows Firewall можа папрасіць дазвол (націсніце Разрэшыць)
  - Пераканайцеся, што іншыя прылады ў той жа сетцы
  - Праверце IP-адрас вашага камп'ютара ў сеткавых наладах

vMix паказвае пусты оверлей:
  - Пераканайцеся, што сервер працуе (праверце акно кансолі)
  - Паспрабуйце адкрыць URL непасрэдна ў браўзеры спачатку
  - Праверце, што "Transparent Background" уключана ў vMix

Для больш падрабязнай інфармацыі гл. BUILD.md або README.md
